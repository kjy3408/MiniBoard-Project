# MiniBoard (게시판)

## 기능
* 회원가입, 로그인 /본인확인 질문으로 ID/PW 찾기
* 게시글 읽기, 쓰기
* 게시글 검색
* 마이페이지 -> 본인이 쓴 게시글 수정/삭제
* 마이페이지 -> 본인이 최근에 본 글
* 마이페이지 -> 내 정보 수정(비밀번호 변경)
* 조회수 증가(조회수 중복 불가)
* 댓글(수정/삭제), 대댓글(삭제) -> 본인이 작성한 댓글만 가능
* 게시글, 댓글 수정 시 수정여부 확인 가능
* 게시글 페이징
